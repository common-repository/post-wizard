== Schnell und einfach Blogbeiträge erstellen - mit künstlicher Intelligenz ==

Das Wordpress-Plugin “Post Wizard” spart viel Zeit und Mühe beim Schreiben von Blogbeiträgen. Post Wizard ermöglicht es Nutzern, automatisch Blogbeiträge zu erstellen, indem der Titel und die SEO-relevanten Keywords vorgegeben werden. Das Plugin generiert durch die Schnittstelle zu GPT-3 von OPEN AI den kompletten Inhalt des Beitrags, unter Berücksichtigung der entsprechenden SEO-Keywords. 
Post Wizard ist eine große Erleichterung für alle Blogger, Content Creatoren und Website-Betreiber bei der Erstellung von Texten und Inhalten. Es ist jedoch wichtig, dass die von GPT-3 generierten Texte nicht unverändert veröffentlicht werden sollten. Um Probleme mit Suchmaschinen wie Google zu vermeiden, empfehlen wir, die Texte noch einmal sorgfältig durchzulesen und anzupassen, bevor sie veröffentlicht werden. Es sollte auch dazu beigetragen werden, dass die Inhalte als einzigartig und von menschlicher Hand geschrieben wahrgenommen werden. 
Prinzipiell ist zu beachten, dass wir als Agentur nicht für eventuelle negative Auswirkungen auf das Ranking Ihrer Website in Suchmaschinenergebnissen verantwortlich gemacht werden können, die durch die Verwendung des Plugins entstehen können. Es ist wichtig, das Plugin verantwortungsvoll zu verwenden und sicherzustellen, dass die generierten Texte entsprechend auf Ihren Content angepasst sind, bevor sie veröffentlicht werden. 
 
Um Post Wizard zu testen, wird ein OPEN AI API-Key benötigt. Hierzu muss ein Firmen-Account bei OPEN AI angelegt werden. 

Noch keinen OPEN AI API Key?
<a href="https://lights-on.io/blog/openai-api-key" target="_blank">Hier</a> haben wir für Sie eine Anleitung erstellt.

Sollten Sie keinen eigenen Firmen-Account bei OPEN AI haben oder wollen das Plugin erstmal nur für ein paar Tage testen, dann melden Sie sich gerne bei uns.
Wir stellen Ihnen einen API-Key zur Verfügung. 

Schreiben Sie uns eine Anfrage an die wordpress@lights-on.io

== Frequently Asked Questions ==

= Wie kann ich Post Wizard installieren? =

Es gibt zwei Wege, wie Sie Post Wizard installieren können.

Sie können das Plugin innerhalb Ihres WordPress Dashboards installieren:
1. Navigieren Sie im WordPress Dashboard zu -> Plugins -> Installieren
2. Suchen Sie nach "Post Wizard LightsOn"
3. Installieren und aktivieren Sie "Post Wizard" von Ihrer Plugin-Seite aus.

Oder Sie installieren das Plugin direkt von WordPress.org
1. Klicken Sie hier oben auf der Seite auf "Download"
2. Uploaden Sie den entpackten Ordner "post-wizard" in den ‘/wp-content/plugins/’ Ordner mit Hilfe von (ftp, sftp, scp, etc…)
3. Installieren und aktivieren Sie "Post Wizard" von Ihrer Plugin-Seite aus.

= Mir wird der "Post Wizard"-Button über den Beiträgen nicht angezeigt =

Das liegt vermutlich daran, dass Sie noch keinen OPEN AI API-Key in dem Plugin hinterlegt haben.
Den API-Key können Sie einfach in Ihrem Dashboard unter "Post-Wizard" hinzufügen
